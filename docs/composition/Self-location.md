# Self-location


###

- Lessons late in the course should be written as if the learner has done what comes before.
    - If the learner does not understand established concepts, we should make sure to supply more support, in the form of practice problems or extra lessons. But we should not write lessons as if the learner will enter the course in the middle.
    - Build on the pedagogy of previous lessons — don't try to pioneer new analogies or frameworks for things that can be handled with what's been laid out.
    - Make sure to return to the conceptual outline and point out problems or things that are missing.
    - If you write a lesson based on something that's ambiguous in the outline, make sure to include your new assumptions at the top of the lesson sketch.
- Some of the lessons sketches string together substantial intermediate results that get used later, but the lesson is written as if I can see the entirety of the lesson.
    - Information that's required for the question at hand should not require scrolling further back than the previous problem. If you need to reference information that isn't a current object of discussion — re-contextualize it before diving in.
    - If you're invoking a formula that isn't already a current object of discussion, then recall it in your writing.
