# Adverbs and Adjectives

The modern world has dissolved common meaning from qualitative adjectives and adverbs. Words like "quickly", "big", or "loud" don't have well understood meanings without familiarity with the speaker. This makes them poorly suited for Brilliant where we need to be precise and accurate.

Avoid using qualitative adverbs for anything but relative comparisons. Absolute statements should never be made using a qualitative adverb or adjective.

!!! success "Yes"
    The particle moves very quickly through the synchrotron.

!!! failure "No"
    Protons are heavier than electrons.
