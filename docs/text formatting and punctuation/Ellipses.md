# Ellipses

!!!tldr "Rule"
    Ellipses on Brilliant usually mean `and so on`, and are valid end punctuation.

Ellipses indicate an omitted word or phrase from a sentence, but this usage is rare in Brilliant courses outside of character dialogues.

An ellipsis is also shorthand for `and so on` in a list. This usage is fine in the math setting by way of the built-in command `\dots`

!!! success "Example"
    The centers of the blocks in the tower follow a harmonic progression: $1/2$ , $1/4$, $1/6$, $1/8$, $\dots$

Note that there is no period following the `\dots`. It is clear that the sentence ends after the list. Including an extra period after the ellipsis does not provide additional clarity, so it is recommended to omit it.

For the sake of clarity, refrain from using an ellipsis in a list of words. It is better in this situation to use phrases like `and so on`, `and more`, or `and others`.

!!! success "Example"
    Along the way, we'll encounter spacetimes with different properties — Ernst spacetime, Friedman-Robertson-Walker, Bertotti-Kasner, and others.

</aside>

Ellipses are occasionally used to convey anticipation or uncertainty. For example,

!!! failure "Example"
    We've seen that the derivative of an object's position is its velocity, but other situations where derivatives arise naturally are on the horizon...

The Brilliant narrator is a trusted, confident voice that promotes learning, so we recommend against this usage outside of character dialogues.
