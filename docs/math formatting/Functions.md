# Functions


<aside>
📏 Function notation is a pain point for learners. Nonetheless, it is used frequently across the site.

</aside>

Outside of a course on calculus, an advanced course that deploys techniques from calculus, or a course that is calculus-adjacent, the function notation $f(x)$ is probably unfamiliar to a typical learner.

If you use function notation, you will need to describe how to interpret it and address misconceptions (for instance, the parentheses don't mean $f$ is multiplying $x$).

One of the benefits of function notation is it makes explicit how one quantity depends on another. Reminding learners of dependencies among quantities is often why we use function notation in a lesson.

At this time, we have little guidance on when function notation is the correct tool in particular situations. Use your common sense. If you are using it in one or two lessons in a course, it is probably not worth the overhead cost of introducing it.
